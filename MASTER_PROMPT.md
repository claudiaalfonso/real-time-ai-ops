# ENERA AI Command Center — Master Prompt

## 1. First Principles (How Sierra AI Thinks)

Before features, we define the rules:

### Latency = Trust
Fast responses are not a performance metric — they are the product.

### Outcomes over Interactions
We don't measure calls. We measure problems solved.

### AI Should Explain Itself
The dashboard must expose why the AI succeeded or failed.

### Escalation Is Signal, Not Failure
Every escalation teaches the system how to improve.

### The Dashboard Must Think
Not just display data — recommend actions.

---

## 2. Dashboard Architecture (Mental Model)

The dashboard operates across three cognitive layers:

### Layer 1 — What is happening right now?
Real-time operational visibility.

### Layer 2 — Is the AI doing a good job?
AI quality, confidence, and performance.

### Layer 3 — What should we do next?
Optimization, coaching, and decision-making.

## 4. Core Modules (Where We're Truly Different)

### 4.1 Call Genome (Not a Call Log)

Each interaction is represented as a call genome, not a row in a table:

**Problem Vector**
(charger offline, billing confusion, range anxiety, firmware issue)

**AI Reasoning Confidence**
"93% confidence in root cause detection"

**Latency Breakdown**
- Speech-to-text
- Reasoning
- Text-to-speech

**User Stress Curve**
Sentiment evolution throughout the call

All of this opens in a side panel, keeping the user in flow.

### 4.2 Resolution Funnel (AI-first, Human-last)

A dynamic funnel showing how problems actually flow:

```
All Calls
 ├─ Resolved by AI (68%)
 │   ├─ First answer (41%)
 │   └─ After clarification (27%)
 ├─ Escalated to Human (7%)
 │   ├─ Hardware replacement
 │   └─ Account edge cases
 └─ Hung Up (25%)
     ├─ Short confusion
     └─ Impatience / noise
```

Each node is interactive and explains:
- Why escalation happened
- Whether it was avoidable
- What data or tooling was missing

### 4.3 Sentiment as an Economic Signal

Sentiment is not decoration.

We track:
- **Sentiment Delta** — how much better the user felt at the end
- **Negative Sentiment Recovery Rate**
- **Silent Dissatisfaction Detector** (Neutral calls with churn risk patterns)

This turns emotion into a business metric.

## 5. The Killer Feature — AI Coaching Layer

Each recommendation includes:
- Estimated impact

This transforms the dashboard from reporting → control system.

## 6. Escalation Intelligence (Not Failure Reporting)

Escalations are analyzed as learning events:

For every escalation:
- Was it necessary?
- Was it too late?
- Could AI have solved it with:
  - Better context?
  - More permissions?
  - Better tooling?

**Key metric:** Unnecessary Escalation Rate

This is the metric executives actually care about.
